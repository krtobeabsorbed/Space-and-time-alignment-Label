### 1.时间空间对齐代码


1. 克隆本项目：

   Bash

   ```
   git clone [https://github.com/krtobeabsorbed/Space-and-time-alignment-Label.git](https://github.com/krtobeabsorbed/Space-and-time-alignment-Label.git)
   cd Space-and-time-alignment-Label
   ```

2. 安装依赖库：

   **推荐 Python 版本**: 3.8.x

   Bash

   ```
   pip install -r requirements.txt
   ```
